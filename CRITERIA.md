# AI Product Tracker - 收录标准 v2.0

## 核心价值
帮创业者/开发者发现值得关注的AI产品，省去筛选时间。

## 分类维度

### 按技术属性
- 🔓 **开源** - 可自部署
- 🔒 **闭源** - SaaS/商业

### 按商业模式
- 💰 **SaaS** - 直接变现
- 🛠️ **工具** - 开发者/创作者工具
- 🔌 **API** - 基础设施

### 按用户群体
- 👨‍💻 **开发者** - 给程序员用
- 👤 **普通用户** - 给终端用户
- 💼 **创业者** - 找灵感/投资

## 收录标准

| 来源 | 最低热度 | 必须标注 |
|------|----------|----------|
| X | 200+ likes | 为什么火 |
| GitHub | 2000+ stars | 做什么的 |
| Product Hunt | top 5 | 创新点 |

## 每期只推荐 3-5 个

每个产品必须有：
1. **一句话点评** - 为什么值得注意
2. **标签** - 分类
3. **热度来源** - 数据支撑
4. **我的判断** - 🔥会火 / 💡创新 / 💰变现
